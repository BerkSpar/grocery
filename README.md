## Key Features

- Create and manage shopping lists
- Real-time cart with price tracking
- Barcode scanner for quick product lookup
- Emoji suggestions for list items
- Timer to track your shopping duration
- Multi-language support (English & Portuguese)
- Offline-first with local database

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com), [Flutter](https://flutter.dev) and [CocoaPods](https://cocoapods.org) (for iOS) installed on your computer.

```bash
git clone https://github.com/BerkSpar/grocery.git
cd grocery
flutter pub get
flutter run
```
